All functions: pure, stateless, Decimal arithmetic only. Proper docstrings on all public functions (QUAL-05). No float anywhere.

Implement `src/fathom/inflation.py`:
- `present_value(future_value: Decimal, annual_inflation: Decimal, month: int) -> Decimal` -- PV = FV / (1 + r_monthly)^month where r_monthly = annual_inflation / 12. Handle month=0 (returns future_value unchanged).
- `discount_cash_flows(payments: list[Decimal], annual_inflation: Decimal) -> list[Decimal]` -- applies present_value to each payment at its month index. Returns list of discounted values.
- `compute_inflation_adjustment(nominal_payments: list[Decimal], annual_inflation: Decimal) -> Decimal` -- returns total nominal minus total discounted (the inflation adjustment amount). Positive means inflation reduces real cost.

Implement `src/fathom/tax.py`:
- `compute_tax_savings(interest_payments: list[Decimal], marginal_tax_rate: Decimal) -> Decimal` -- sum of interest_payments * marginal_tax_rate. Returns Decimal("0") if rate is zero. Quantize final result to cents.

All functions: pure, Decimal only, proper docstrings.
