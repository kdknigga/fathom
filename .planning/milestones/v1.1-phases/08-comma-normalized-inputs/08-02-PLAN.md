<task type="auto">
  <name>Task 1: Create formatting.js with delegated event handlers</name>
  <files>src/fathom/static/formatting.js, src/fathom/templates/base.html</files>
  <action>
    1. Create `src/fathom/static/formatting.js` as an IIFE with three delegated event listeners on the form element (`document.getElementById('comparison-form')`). If form is null, return early (safety for pages without the form).

    **focusin handler:** When a `data-monetary` input receives focus, strip commas from the value: `e.target.value = e.target.value.replace(/,/g, '')`. This lets the user edit a raw number without cursor position issues.

    **focusout handler:** When a `data-monetary` input loses focus:
    - Strip commas from value: `var raw = e.target.value.replace(/,/g, '')`
    - If raw is empty or NaN, return (don't format invalid input)
    - Split raw on '.' to detect decimal places
    - Use `parseFloat(raw).toLocaleString('en-US', {...})` to format
    - If decimal part exists, set minimumFractionDigits and maximumFractionDigits to the length of the decimal part (preserves trailing zeros per locked decision)
    - If no decimal part, use plain `toLocaleString('en-US')`

    **paste handler:** When pasting into a `data-monetary` input:
    - `e.preventDefault()`
    - Get clipboard text via `(e.clipboardData || window.clipboardData).getData('text')`
    - Strip `$`, commas, and whitespace: `text.replace(/[$,\s]/g, '')`
    - Set `e.target.value` to the cleaned text
    - Commas will be added on subsequent blur (consistent with blur-only formatting rule)

    Use event delegation (listeners on form, check `e.target.hasAttribute('data-monetary')`) — NOT per-element listeners. This automatically covers HTMX-swapped elements.

    2. Add a `<script>` tag to `base.html` to load formatting.js. Place it at the end of the body, after the HTMX script. Use Flask's `url_for('static', filename='formatting.js')` pattern consistent with existing vendor asset loading.
  </action>
