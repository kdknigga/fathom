    1. Modify `_try_decimal()` in forms.py (line 44): After the empty check, add `cleaned = value.strip().replace("$", "").replace(",", "").replace(" ", "")`, then check if cleaned is empty, then `Decimal(cleaned)`. This is the single chokepoint for all Decimal parsing.

    2. Modify `_to_money()` in forms.py (line 543): Same stripping pattern as _try_decimal. Add `cleaned = value.strip().replace("$", "").replace(",", "").replace(" ", "")` before Decimal conversion.

Change `return v` to strip commas/dollars/spaces from v before returning: `return v.strip().replace("$", "").replace(",", "").replace(" ", "")`. Apply the same pattern to all OptionInput field validators that return raw strings which later get passed to `Decimal()` or `_to_money()`.

    5. Create `src/fathom/formatting.py` with a `comma_format(value: str) -> str` function:

       - If the Decimal equals its integral value AND the original had no decimal point, format as `f"{int(dec):,}"`
       - Otherwise, use the Decimal's string representation split on `.`, format integer part with `f"{int_part:,}"`, rejoin with decimal part to preserve exact decimal places as entered
       - This preserves trailing zeros (25000.10 -> 25,000.10)
